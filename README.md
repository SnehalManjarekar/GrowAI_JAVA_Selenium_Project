# GrowAI JAVA Selenium Project

This is an automation testing project developed using:

- Java
- Selenium WebDriver
- TestNG
- Maven
- IntelliJ IDEA

## Project Features

- Login Test Automation
- Valid Login Test
- Invalid Login Test
- Blank username & password Login Test
- Selenium WebDriver Setup
- TestNG Assertions
- Maven Project Structure

## Tools & Technologies

| Tool | Purpose |
|------|---------|
| Java | Programming Language |
| Selenium | Browser Automation |
| TestNG | Testing Framework |
| Maven | Dependency Management |
| IntelliJ IDEA | IDE |


## Project Structure

src
├── main
│   └── java
├── test
├── pom.xml
└── README.md

## How to Run

1. Clone the repository
2. Open project in IntelliJ IDEA
3. Install Maven dependencies
4. Run the Main class

Example:
- Right click Main.java
- Click Run 'Main.main()'
